Beginning CSS

Explain the difference between inline, embedded, and external CSS.

— The difference is Inline CSS and Embedded CSS are in the HTML file, or inside the HTML, but External CSS is a different file where you can edit or make different styles in multiple pages

Explain why you should only use external CSS.

— The reason why we should only use external CSS is because it makes HTML clean, or easier to update or edit, and not just one page but multiple pages.

What is a selector used for?

— The Selector is used for styles or elements that you are going to use in the HTML
