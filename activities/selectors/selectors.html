Engagement

  1. List out the different ways to select item.
  2. What is the difference between ID and a Class?
  3. What is the difference between SPAN and a DIV tag?
  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?
  5. What are the different ways to select a color?

Answers

  • An item can be selected using element selectors, ID selectors, class selectors, attribute selectors, or pseudo-selectors.
  • An ID is unique and used for one element, while a class can be reused for multiple elements.
  • A span is an inline element used for small content, while a div is a block-level element used to group larger sections.
  • A div is a generic container, while header, main, and footer are semantic tags that describe the purpose of the content.
  • Colors can be selected using color names, HEX values, RGB/RGBA, and HSL/HSLA.
