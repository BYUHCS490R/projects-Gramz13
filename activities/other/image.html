Questions

  1. What kind of photos are good to encode using JPEG?
    Realistic images, nature scenes, photographs of people, and social media photos.
  2. What kind of photos are good to encode using GIF?
    Simple graphics with few colors, logos and icons, and cartoons or illustrations.
  3. What is unique about the PNG format?
    It is good for logos, and transparency.
  4. How do you resize your photo to the size it should be displayed?
    You can use pixlr, or other photo editor to resize images.
  5. Why should you resize photos using photo editing software instead of resizing it using CSS?
    You should resize photos using photo editing software because CSS resizing still downloads the full-size image, which slows loading time and wastes bandwidth.
  6. What is CSS validation?
    CSS validation is the process of checking CSS code for errors and ensuring it follows official CSS standards so it works correctly across different browsers.
