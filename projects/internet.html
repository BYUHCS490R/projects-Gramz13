Internet

The Internet is a global network of computers and cables that allows them to communicate with each other. It uses Internet Protocol to communicate between networks and devices. Lastly, computers have their own IP address, and nowadays the way to access internet are through underground cables and satellites.

Web

Web or commonly known as The World Wide Web (WWW) is an information systems that enables content sharing, it is also a global system of interconnected files, documents and resources such as web pages, images, videos that can access by the users using the Internet.

How are the Internet and Web different?

Internet and Web are technically different. Internet connects computers and devices around the world while the Web is what people use on the internet to visit websites and find information.

ASCII and Unicode

ASCII and Unicode are systems used to represent text in computers. ASCII is an old system that uses numbers to represent basic English letters, numbers, and symbols. While on the other hand, Unicode is the newer system that represent characters from many languages around the world as well as emojis and special symbols.

What is UTF-8?

UTF-8 is a Unicode encoding that communicates to the computers how to store and display text. It supports characters from many languages, uses less space for common characters, and works well with the old system ASCII.

URL

URL (Uniform Resource Locator) is the web address uses to find resource on the internet, such as website, image, or video.

Domain Name

A domain name is the easy-to-read name used to identify a website on the internet.

HTTP

HTTP (HyperText Transfer Protocol) is a set of rules that lets your browser and a website communicate with each other. It allows the user to view web pages and move from one page to another.

HTTPS

HTTPS (HyperText Transfer Protocol Secure) is like HTTP but safer. It encrypts the information sent between users browser and a website, keeping the users data private and protected.

IP Address

IP Address (Internet Protocol address) is a unique number assigned to every device connected to the internet. It is like the device's digital home address, allowing computers to find and communicate with each other.

DNS

DNS or Domain Name System is like the internet's phonebook. It translates website names into IP addresses so computers can find and connect to them.

W3C Recommendations

W3C Recommendations are like the "rules" for making website. They help web developers build websites that work well and look right on any computer or browser.

XML, XHTML, HTML, HTML5

XML is a way to organize and store data so computers can read it easily. XHTML is a more strict version of HTML that follows rules to make web pages work better. HTML is the basic language used to create and structure web pages. HTML5 is the latest or newest version of HTML with extra features for multimedia, apps, and modern websites.
